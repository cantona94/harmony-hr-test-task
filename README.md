## App: Next JS - HarmonyHR test-task

### Run on a local host

1. Clone the repository

```sh
git clone https://github.com/cantona94/harmony-hr-test-task.git
```

2. Install NPM packages

```sh
cd harmony-hr-test-task
```

```sh
npm install
```

3. Launch of the project

```sh
npm run dev
```
